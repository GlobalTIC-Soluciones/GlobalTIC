# Documentos

Cada integrante debe subir un commit de lo que quiera.

comandos de git
```shell
git init

git add .

git commit -m "first commit"

git remote add origin https://github.com/NOMBRE_USUARIO/NOMBRE_PROYECTO.git

git push -u origin master
```
